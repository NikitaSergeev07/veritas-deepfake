## Слайд 2: Проблема и обоснование

**Дипфейки — растущая угроза цифровому доверию**

- Количество дипфейков в интернете увеличилось на 900% за последние 3 года
- В 2025 году зафиксировано более 500 000 дипфейк-видео и аудио
- Дипфейки используются для: мошенничества (поддельные звонки от «руководства»), политической дезинформации, шантажа и компрометации, подделки цифровых документов

**Проблема:** существующие инструменты обнаружения дипфейков либо платные, либо работают только с одним типом контента, либо требуют отправки файлов на внешние серверы.

**Наше решение:** единый бесплатный веб-сервис, который анализирует изображения, аудио и видео локально на устройстве пользователя.

## Слайд 4: Основная идея

**Veritas** — единый веб-сервис для проверки файлов на дипфейки

Ключевые принципы:
- **Мультимодальность**: поддержка изображений, аудио и видео в одном интерфейсе
- **Приватность**: весь анализ выполняется локально, файлы не покидают устройство
- **Ансамблевый подход**: комбинация нескольких AI-моделей для повышения точности
- **Простота**: drag & drop интерфейс, результат за секунды

## Слайд 5: Функционал прототипа

### Модуль 1: Детекция дипфейков в изображениях
- Модель CLIP ViT-B/16 + обученная MLP-голова
- Анализ с 3 ракурсов (полный кадр, центральный кроп, фокус-кроп)
- Агрегация оценок → вердикт с уверенностью

### Модуль 2: Детекция дипфейков в аудио
- 5 предобученных Wav2Vec2 моделей
- Стекинг через LogisticRegression (meta-learner)
- Точность 98.2% на тестовом датасете
- Автоматическая конвертация M4A/OGG → WAV через ffmpeg

### Модуль 3: Детекция дипфейков в видео
- VideoMAE — temporal transformer (86M параметров)
- Обучен на FaceForensics++ C23
- Автоматическая детекция лица через MediaPipe + кроп
- Посегментный анализ длинных видео с таймлайном

### Общие возможности
- Drag & drop загрузка файлов
- Двуязычный интерфейс (RU / EN)
- Детальная визуализация результатов: прогресс-бары, оценки моделей, метаданные

## Слайд 6: Технологии и реализация

### Frontend
- **Vue 3** + TypeScript + Vite
- Минимальный дизайн, вдохновлённый VirusTotal
- Реактивный UI с health-polling бэкенда

### Backend
- **FastAPI** — асинхронный Python-фреймворк
- ThreadPoolExecutor — неблокирующий ML-инференс
- Ленивая загрузка моделей (lazy loading) для экономии памяти

### AI / ML
- **CLIP ViT-B/16** (OpenAI) — извлечение визуальных признаков изображений
- **Wav2Vec2** (5 моделей HuggingFace) — аудио-классификация
- **VideoMAE** (FaceForensics++ C23) — temporal анализ видео
- **MediaPipe** — детекция лиц для видео-кропа
- **scikit-learn** — стекинг ансамбль (LogisticRegression)

### Инфраструктура
- ffmpeg — конвертация аудио/видео форматов
- Поддержка CPU и CUDA GPU
- Поддержка 15+ форматов файлов (JPG, PNG, WAV, MP3, MP4, MOV и др.)

## Слайд 7: Тестирование и обратная связь

### Результаты тестирования

| Модуль | Датасет | Метрика |
|--------|---------|---------|
| Image | Custom train/val split | ~95% accuracy |
| Audio | garystafford/deepfake-audio-detection (1866 семплов) | 98.2% accuracy |
| Video | FaceForensics++ C23, DFDC samples | Корректное определение face-swap дипфейков |

### Выявленные ограничения
- Аудио-модель обучена на ElevenLabs/Kokoro TTS — хуже работает на других типах синтеза
- Видео-модель оптимизирована под face-swap — менее точна для lip-sync дипфейков
- На CPU первый запуск загружает модели 2–5 минут, далее — секунды

### Обратная связь
- Интерфейс интуитивен: drag & drop + один клик для анализа
- Результаты представлены наглядно: вердикт + процент уверенности + детализация по моделям
